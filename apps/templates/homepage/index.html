Charge transfer state energy of organic photovoltaic materials


        Morphology related descriptors is crucial to the prediction of charge transfer state energy. To obtained the morphology related descriptors, several scripts have been written using Tool Command Language (TCL). If you want to obtain the effective distance defined as the minimum distance between different unit of two molecular, inclination angle, crossing angle, backbone length and width of polymer, you can run the script we provide.
